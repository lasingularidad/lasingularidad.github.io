lasingularidad.es

En este proyecto contamos como hemos construido esta misma página web (lasingularidad.es). Se trata de una página estática, creada con el gestor de contenidos Lektor y alojada en GitHub Pages.

Teniamos dos requisitios fundamentales: Poder usar un servicio de alojamiento gratuito; y disponer de una interfaz de gestion para poder ir añadiendo contenido facilmente.

  • Sumario

    Vision general de proyecto y de sus componentes principales.

  • Preparación

    Preparación y configuración de los elementos necesarios para empezar a trabajar en nuestra página web. Tanto el alojamiento como el entorno de desarrollo que usaremos luego.

  • Lektor

    Descripción de la estructura del proyecto en Lektor y detalles de implementación de las funcionalidades añadidas.

  • CSS

    Personalización de la apariencia de la página mediante mediante CSS
